Visão Geral do Suporte He-net

Conceitos fundamentais para o atendimento

A He-net é uma provedora de internet por fibra óptica. O suporte técnico e o call center são responsáveis por atender clientes que enfrentam problemas de conectividade, lentidão, configurações ou questões financeiras.

1.1 Conceito de Viabilidade

Viabilidade é a verificação de que a He-net já possui cobertura/atendimento no endereço do cliente. Só é possível instalar o serviço onde há infraestrutura disponível.

1.2 Principais Solicitações dos Clientes

🔴 Problemas Técnicos
  • Sem conexão / Fibra rompida
  • Falha na rede
  • Lentidão
  • Intermitência (quedas na conexão)
🔵 Questões Administrativas
  • Questões financeiras (boleto, plano)
  • Informações sobre planos
  • Mudança de titularidade
  • Cancelamento de serviço

1.3 Causas Comuns de "Sem Conexão"

  • Aparelho com fonte queimada
  • Roteador travado (necessita reinicialização)
  • Cabos conectados em porta incorreta
  • Roteador resetado — necessita reconfiguração
  • Ficha rompida / problema no conector
💡
Dica para novatos Sempre confirme o tipo de equipamento do cliente (TP-Link, Nokia, Huawei, etc.) antes de orientar qualquer procedimento. Cada marca tem comportamentos diferentes.

5.4 IXC/Radius Desconectado — Procedimento

Quando o PPPoE não aparece ou a conexão cai com IXC/Radius desconectado, escreva no protocolo: "IXC/Radius desconectado" — isso salva a senha novamente no sistema.

1

2
Dar restore na UNM e pedir ao cliente para reiniciar o roteador
3
Se for TP-Link e o cabo estiver "linkando": verificar VLAN
4
Se o cliente for de Itabuna com IP Público habilitado: desabilitar o IP Público — pode resolver

Conhecendo o Roteador

Fundamentos de hardware, portas, frequências e desempenho

7.1 O que é um Roteador

O roteador é um equipamento Indoor (uso interno), responsável por:

  • Converter o sinal cabeado (tecnologia 802.3) para Wi-Fi (802.11) — essa conversão gera uma pequena elevação no ping
  • Definir as melhores rotas para o tráfego de dados na rede local
  • Aumentar a quantidade de portas Ethernet disponíveis na rede
  • Compartilhar mídias em rede via porta USB/LAN (em modelos compatíveis)
⚠️
Limitação do uso residencial Roteadores Home (residenciais) não suportam grandes larguras de banda nem alto número de dispositivos simultâneos. Não são adequados para cobrir eventos com muitas pessoas.

7.2 Portas WAN e LAN

🌐 WAN — Wide Area Network

Porta destinada ao link externo. Faz a comunicação com a internet. É onde se conecta o cabo vindo da ONU. Representa a rede de longo alcance que pode cobrir países ou continentes.

🏠 LAN — Local Area Network

Portas destinadas à distribuição do link internamente. É onde se conectam computadores, notebooks, etc. Roteadores convencionais têm geralmente 4 portas LAN + 1 WAN.

7.3 Taxa de Transmissão Wi-Fi e Impacto no Plano

📌
Como interpretar o modelo do roteador Exemplo: TL-WR340G54M → taxa máxima de 54 Mbps (protocolo G). Mesmo contratando 500 Mbps, o Wi-Fi ficará limitado a 54 Mbps. A taxa indicada é a soma do download + upload — logo, o download real será em torno de 50% desse valor.
Transmissão do roteadorMáximo real via Wi-Fi (aprox.)Plano que suporta bem
54 Mbps (protocolo G)~24 Mbps downloadAté 20 Mbps
150 Mbps~70 Mbps downloadAté 60 Mbps
300 Mbps (2 antenas)~150 Mbps downloadAté 150 Mbps

7.4 Antenas e Capacidade de Transmissão

Quanto mais antenas, maior a capacidade de transmissão e a vazão de dados. Exemplo: roteador de 300 Mbps com 2 antenas transmite 150 Mbps em cada antena.

7.7 Botões do Roteador — Quando Usar

BotãoSituação de UsoProcedimento
POWER IXC/Radius desconectado + alarme DYINGGASP na UNM + todos cabos conectados + nenhuma luz no aparelho Solicitar que o cliente pressione o botão Power até que as luzes retornem. Se não funcionar, enviar técnico.
RESET Apenas para TP-Link ou Intelbras. Após verificar WAN-VLAN, cabo linkando e conectado, IXC/Radius desconectado e UNM online Auxiliar o cliente a realizar o reset. Ver procedimento detalhado na seção 7.8.

7.8 Procedimento Completo de Reset do Roteador (TP-Link)

1
Verificar se o cliente tem acesso à parte de baixo do aparelho (onde fica a senha padrão de fábrica)
2
Solicitar que pressione o botão Reset (User) com auxílio de um palito ou agulha, até as luzes apagarem/acenderem
3
Aguardar aparecer um novo nome de rede — ex.: rede padrão TP-LINK (quando resetado, o roteador exibe o nome padrão de fábrica)
4
Conectar à rede usando a senha PIN impressa no fundo do aparelho (sequência numérica)
5
Acessar a interface via celular ou computador, no navegador (Google Chrome): 192.168.0.1 ou 192.168.0.2
6
Inserir a senha de acesso ao painel. Senhas padrão: henet9700 | admin | 64312600
7
Configurar: usuário e senha PPPoE, nome e senha da rede Wi-Fi (2.4G e 5G), DNS primário e secundário
8
Verificar se foi salvo na UNM após configurar no Flashman
🔑
Senhas padrão do roteador TP-Link (He-net) henet9700  |  admin  |  64312600
Use essas senhas somente após reset. Nunca compartilhe com clientes externos.

7.9 Identificação "A1" no Nome do Roteador

💡
O que significa "A1" no final do nome do equipamento? Indica que o cliente possui um Roteador TP-Link + uma ONU (conversor de fibra). A cadeia de conexão é:

Fibra óptica → ONU → Cabo de rede → Roteador TP-Link

PPPoE e Configurações de Conexão

Protocolo de autenticação, credenciais e configuração correta

9.1 O que é PPPoE

PPPoE = Point-to-Point Protocol over Ethernet. É o protocolo de autenticação que conecta o roteador do cliente ao concentrador PPPoE do provedor He-net.

💡
Fluxo de conexão PPPoE Provedor (data center / concentrador PPPoE) → ONU em modo Bridge → Roteador (faz autenticação com login e senha PPPoE) → Dispositivos do cliente conectados

9.2 Padrão de Novo PPPoE

📌
Como criar novo PPPoE para o cliente
  • Login: Nome do titular em minúsculo + 3 primeiros dígitos do código do cliente. Ex.: claudiane320
  • Senha: Código completo do cliente. Ex.: 320172
Após criar, configure no Flashman E verifique na UNM se foi salvo.

9.3 Ordem de Configuração — Regra Importante

🔴
Sempre configurar nesta ordem:
  1. Configure primeiro no Flashman
  2. Depois verifique se foi salvo na UNM
Esta regra se aplica tanto para lentidão quanto para troca de senha. Nunca inverta a ordem.

9.4 IP Público — Quando Habilitar / Desabilitar

  • Habilitar IP Público: quando o cliente não consegue conectar câmera de segurança à rede via Wi-Fi (junto com desabilitar Port Isolation)
  • Desabilitar IP Público: quando cliente de Itabuna está sem conexão com IXC/Radius desconectado e UNM online

9.5 TVBOX e Serviço de Canais

📺
TVBOX Pirata — Não Homologado TVBox pirata (não homologado pela He-net) não tem suporte técnico oficial. Se o cliente relata problema no aparelho de canais, o procedimento é: Port → Apagar → Criar Novo (Unicast / Multicast / 300 / Tag).

9.6 Confirmação de Serviço

Ao confirmar o serviço do cliente, sempre faça a troca de senha/usuário como parte do processo de ativação.

Casos Práticos de Atendimento

Roteiros para as situações mais comuns do dia a dia

10.1 Cliente sem Wi-Fi no celular

1
Solicitar ao cliente que "esqueça" a rede Wi-Fi no dispositivo e tente conectar novamente
2
Se não resolver: dar Restore na UNM e reiniciar o roteador

10.2 Rede 5 GHz não aparece

1
Na UNM: desabilitar e habilitar a rede 5 GHz (Wifi Service Config)
2
Se não resolver: criar uma rede "teste" e solicitar ao cliente que verifique se aparece
3
Verificar se as redes estão unificadas pela função Easy Mesh (rede 5G sem o "5G" no nome) — se sim, desabilitar e separar as redes

10.3 Redes Wi-Fi não aparecem para nenhum dispositivo

1
Verificar na UNM se as redes estão desabilitadas ou ocultas
2
Se for TP-Link: solicitar ao cliente que remova o aparelho da tomada e aguarde 30 segundos
3
Se for Nokia: pressionar o botão Wi-Fi no aparelho
4
Demais aparelhos: pressionar o botão WPS

10.4 Problemas na rede 2.4 GHz (modelo HGD3)

⚠️
Aparelho HGD3 com rede 2.4G instável Se o cliente relata que a rede 2.4G aparece e some (enquanto 5G está normal) e o aparelho for modelo HGD3: será necessária troca do equipamento. Não informe ao cliente que é troca — apenas oriente que um técnico precisará verificar.

10.5 Cliente não consegue conectar impressora

Para conectar impressora à rede 2.4G: habilitar o botão WPS via interface do aparelho (necessário para autenticação da impressora na rede Wi-Fi).

10.6 Cliente não consegue conectar câmera de segurança

1
Liberar o IP Público do cliente
2
Desabilitar o Port Isolation na UNM

10.7 Intermitência (quedas repetidas)

1
Verificar no IXC (CPF/CNPJ do cliente → relatório de intermitência) as quedas registradas
2
Realizar as configurações necessárias na UNM/Flashman
3
Solicitar ao cliente que monitore e informe se o problema continua

10.8 Sem Conexão — IXC/Radius Desconectado e UNM Online (TP-Link)

1
Verificar se o cabo está linkando
2
Verificar a VLAN do equipamento
3
Se cliente for de Itabuna e tiver IP Público habilitado: desabilitar IP Público

10.9 Aparelho Fiberhome — Acesso ao Admin

1
Na UNM: ir em Port Service → ONU Local Manage
2
Habilitar as caixas que estão desabilitadas
3
No Flashman: buscar o MAC do equipamento → acessar Credenciais da Interface para obter admin e senha

Financeiro, Cancelamento e Retenção

Procedimentos para situações de cancelamento e retenção de clientes

11.1 Verificação de Inadimplência

Para verificar se há fatura em aberto ou em atraso: no IXC Soft, acesse a aba Financeiro — faturas em atraso aparecerão em vermelho.

11.2 Plano Suspenso

Quando o cliente tem plano suspenso por inadimplência, encaminhar para o setor Financeiro (ramal *2 3001#).

11.3 Cancelamento — Regras

🔴
Regras críticas para cancelamento
  • O cancelamento só pode ser feito com o titular do contrato
  • Verificar o período contratual de 12 meses e calcular o valor rescisório proporcional aos meses restantes
  • NUNCA coloque seu nome no campo de cancelamento — use apenas "cancelamento" como identificação
  • Ao transferir para o setor de Cancelamento, sempre explique o motivo antes — nunca transfira sem contextualizar

11.4 Retenção — Argumentos e Estratégias

Motivo do CancelamentoEstratégia de Retenção
Mudança de endereço para cidade sem coberturaVerificar troca de titularidade · Confirmar se a mudança é definitiva · Perguntar destino e data
Problemas técnicos recorrentesOferecer suporte técnico garantindo solução · Acionar NOC se necessário
Proposta melhor de outro provedorVerificar com o Comercial a possibilidade de contraproposta (ramal *2 3003#)

11.5 Back Office — Confirmação de Atendimento Presencial

Para antecipar ou confirmar visita técnica presencial, acionar o Back Office pelo ramal *2 3004#.

Glossário

Termos técnicos e siglas utilizados na He-net
CPE
Customer Premises Equipment — equipamento instalado na residência do cliente (ONU, roteador, etc.)
DNS
Domain Name System — servidor que traduz nomes de domínio (ex.: google.com) em endereços IP
DYINGGASP
Alarme de falta de energia no equipamento ONU/ONT do cliente
ERP
Enterprise Resource Planning — sistema integrado de gestão empresarial (ex.: IXC Soft)
Fast Ethernet
Padrão de rede 10/100 Mbps, velocidade máxima de 100 Mbps por porta física
Gigabit Ethernet
Padrão de rede 10/100/1000 Mbps, velocidade máxima de 1 Gbps por porta física
Homologado
Equipamento aprovado e certificado para uso na rede He-net
IP
Internet Protocol — endereço numérico que identifica um dispositivo na rede (ex.: 192.168.0.1)
IP Privado
Endereço IP usado dentro da rede local (LAN), não acessível diretamente pela internet
IP Público
Endereço IP único visível na internet, necessário para câmeras e alguns serviços externos
LAN
Local Area Network — rede local que conecta dispositivos dentro de um mesmo espaço físico
LINKLOSS
Alarme de perda de sinal óptico na fibra — indica rompimento, dobra ou problema no conector
MAC Address
Endereço físico único de cada interface de rede, usado para identificar equipamentos
NOC
Network Operations Center — centro de operações de rede, responsável pelo monitoramento e suporte técnico avançado
ONU
Optical Network Unit — equipamento conversor de fibra óptica para sinal elétrico (padrão IEEE)
ONT
Optical Network Terminal — equipamento conversor de fibra óptica para sinal elétrico (padrão ITU-T), geralmente na residência
OS
Ordem de Serviço — registro formal de uma visita técnica ou atividade agendada
Port Isolation
Recurso que isola dispositivos na rede impedindo comunicação entre eles — deve ser desabilitado para uso de câmeras
PPPoE
Point-to-Point Protocol over Ethernet — protocolo de autenticação entre roteador e concentrador do provedor
Radius / IXC
Servidores de autenticação e autorização de acesso à internet — quando desconectado, o cliente perde acesso
RJ45
Conector padrão para cabos de rede UTP (8 pinos), usado em todas as conexões Ethernet
UNM
Sistema Unificado de Monitoramento — plataforma He-net para gerenciar ONUs, configurar Wi-Fi e monitorar conexões
UTP
Unshielded Twisted Pair — tipo de cabo de rede trançado sem blindagem, com conector RJ45
VLAN / WAN-VLAN
Virtual LAN — segmentação virtual da rede. Na He-net, a WAN-VLAN é o ramal do técnico precedido pelo dígito 3
WAN
Wide Area Network — rede de longa distância; porta WAN do roteador é onde se conecta o link externo (ONU)
Wireless / Wi-Fi
Rede sem fio — comunicação por rádio frequência (2.4 GHz ou 5 GHz) entre dispositivos e roteador
WPS
Wi-Fi Protected Setup — botão/função que simplifica a conexão de dispositivos à rede Wi-Fi sem digitar senha
